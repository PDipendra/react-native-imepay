# react-native-imepay

react-native wrapper for imepay. Especially designed for nepal

## Installation

```sh
npm install react-native-imepay
```

## Usage

```js
import Imepay from "react-native-imepay";

// ...

const result = await Imepay.multiply(3, 7);
```
